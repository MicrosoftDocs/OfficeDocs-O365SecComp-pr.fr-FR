# <a name="submit-spam-non-spam-and-phishing-scam-messages-to-microsoft-for-analysis"></a>Soumission des messages indésirables, légitimes ou des tentatives de hameçonnage à Microsoft pour analyse

Il peut être frustrant lorsque les utilisateurs de votre organisation reçoivent du courrier indésirable (spam) ou hameçonnage anti-spam dans leur boîte de réception, ou s’ils ne reçoivent un message électronique légitime car il est marqué comme courrier indésirable. Nous sommes constamment usuelles nos filtres anti-spam pour être plus précis. Vous et vos utilisateurs peuvent aider à ce processus à la soumission de messages faux négatifs et false positif du courrier indésirable à Microsoft pour analyse. Négatif « false » est un message de courrier indésirable qui doit avoir été mais n’était pas identifié comme courrier indésirable. « Faux positif » est un message électroniques légitimes qui a été incorrectement identifié comme courrier indésirable. 
  
> [!NOTE]
> En raison du volume élevé d’envois que nous, nous ne pouvons pas être en mesure de répondre à toutes les demandes pour l’analyse. 
  
## <a name="submit-junk-or-phishing-messages-that-passed-through-the-spam-filters"></a>Soumettre des courriers indésirables ou des tentatives de hameçonnage passés à travers les filtres de courrier indésirable
<a name="sectionSection0"> </a>

Si vous recevez un message passé à travers le courrier indésirable qui filtre et doit être considérée comme indésirable ou un hameçonnage semblant, vous pouvez envoyer le message « faux négatifs » aux équipes de l’analyse du courrier indésirable de Microsoft et Microsoft Phishing Analysis, selon le cas. Les analystes seront passez en revue le message et l’ajouter aux filtres de service à l’échelle s’il répond aux critères de classification. 
  
Pour les paramètres du courrier indésirable supplémentaires qui s’appliquent à toute l’organisation, voir [bloquer le courrier indésirable courrier électronique avec le filtre de courrier indésirable Office 365 pour éviter les problèmes négatifs false](https://go.microsoft.com/fwlink/p/?LinkId=534225). Cet article contient des conseils pour éviter les faux négatifs.
  
Vous pouvez soumettre des courriers indésirables en procédant comme suit :
  
- Pour Outlook et Outlook sur les utilisateurs du web, utilisez le complément de Message de rapport pour Microsoft Outlook. Pour plus d’informations sur l’installation et l’utilisation de cet outil, voir [Activer le complément de Message de rapport](https://support.office.com/article/4250c4bc-6102-420b-9e0a-a95064837676). 
        
- Vous pouvez également utiliser le courrier électronique à envoyer des messages à Microsoft qui doivent être classées comme indésirable ou hameçonnage escroqueries, comme décrit dans la procédure suivante.
    
### <a name="use-email-to-submit-junk-spam-or-phishing-scam-messages-to-microsoft"></a>Utilisation de la messagerie pour soumettre des courriers indésirables (spam) ou des tentatives de hameçonnage à Microsoft
<a name="Useemailtosubmitjunkspamorphishingscammessages"> </a>

Pour soumettre un courrier indésirable ou une tentative de hameçonnage à Microsoft, procédez comme suit :
  
1. Créer un message électronique vide.
    
2. Adressez le message à l’équipe Microsoft qui passe en revue les messages, comme suit : 
    
  - Pour les messages indésirables : junk@office365.microsoft.com
    
  - Pour les messages de phishing anti-spam : phish@office365.microsoft.com
    
3. Copiez et collez le message anti-spam indésirable ou hameçonnage dans le nouveau message comme pièce jointe. 
    
    > [!NOTE]
    > Vous pouvez joindre plusieurs messages vers le nouveau message. Assurez-vous que tous les messages sont du même type — hameçonnage anti-spam ou messages de courrier indésirable. > Quitter le corps du nouveau message vide. 
  
4. Cliquez sur **Envoyer**.
    
## <a name="submit-messages-that-were-tagged-as-junk-but-should-have-been-allowed-through"></a>Soumission de messages marqués comme courrier indésirable mais qui auraient dû être transmis
<a name="sectionSection1"> </a>

Si un message a été incorrectement identifié comme courrier indésirable, vous pouvez envoyer le message « faux positif » à l’équipe d’analyse de courrier indésirable Microsoft. Les analystes sont alors évaluer et analyser le message. Selon les résultats de l’analyse, les règles de filtrage du courrier indésirable à l’échelle du service peuvent être ajustés pour autoriser le message.
  
Les administrateurs peuvent consulter plus d’informations de configuration du courrier indésirable qui s’applique à un ensemble de l’organisation. Découvrez [comment vous assurer qu’un message n’est pas marqué comme courrier indésirable](https://go.microsoft.com/fwlink/p/?LinkId=534224). Cette information est utile si vous avez le contrôle de niveau administrateur et que vous souhaitez empêcher de faux positifs.
  
Vous pouvez envoyer des messages de courrier non indésirable de la manière suivante :
  
- Si vous utilisez l’action **déplacer le message vers le dossier courrier indésirable** lorsque vous configurez vos filtres de contenu (il s’agit de l’action par défaut), les utilisateurs peuvent libérer des messages faux positifs dans leur dossier Outlook ou OWA Junk Email. 
    
  - Les utilisateurs d’Outlook peuvent release messages faux positifs à l’aide de l’option du menu contextuel **Pas indésirable** . Toutefois, ils doivent envoyer le message à Microsoft par e-mail, comme indiqué dans la procédure décrite dans cet article. 
    
  - Les utilisateurs d’OWA peuvent libérer des messages faux positifs et les envoyer à Microsoft pour analyse à l’aide de l’action **Marquer comme légitime** . Pour plus d’informations, voir [courrier indésirable de rapport et de hameçonnage dans Outlook sur le web ](report-junk-email-and-phishing-scams-in-outlook-on-the-web-eop.md).
    
- Si vous utilisez l’action du **message de quarantaine** au lieu de l’action **déplacer le message vers le dossier courrier indésirable** lorsque vous configurez vos filtres de contenu : 
    
  - Les administrateurs peuvent débloquer les messages mis en quarantaine du courrier indésirable et les signaler comme faux positifs à partir du centre d’administration Exchange. Pour plus d’informations, voir [Rechercher et débloquer les messages mis en quarantaine en tant qu’administrateur](find-and-release-quarantined-messages-as-an-administrator.md).
    
  - Les utilisateurs peuvent libérer leurs propres messages mis en quarantaine du courrier indésirable et les signaler comme faux positifs via les canaux suivants : 
    
  - L’interface utilisateur du centre d’administration Exchange (EAC). Pour plus d’informations, voir [Find and Release Quarantined Messages (End Users)](http://technet.microsoft.com/library/e439b560-827a-4807-abd3-6b861c1ff786.aspx).
    
  - Les messages de notification de courrier indésirable pour les utilisateurs finaux (s'ils sont activés par votre administrateur). 
    
- Vous pouvez également utiliser le courrier électronique à envoyer des messages à Microsoft qui ne doivent pas être considérées comme du courrier indésirable. Lorsque vous effectuez cette opération, assurez-vous que vous utilisez les étapes de la procédure suivante.
    
### <a name="use-email-to-submit-false-positive-messages"></a>Utilisation du courrier électronique pour soumettre des messages faux positifs

Utiliser la même procédure comme décrit dans le «[utilisation du courrier électronique pour envoyer le courrier indésirable (spam) ou à Microsoft, les messages de phishing anti-spam ](submit-spam-non-spam-and-phishing-scam-messages-to-microsoft-for-analysis.md#Useemailtosubmitjunkspamorphishingscammessages)», mais l’envoyer à not_junk@office365.microsoft.com.
  
## <a name="spam-evaluation-and-rules-deployment"></a>Déploiement d’évaluation et de règles de courrier indésirable
<a name="sectionSection2"> </a>

L’équipe d’analyse du courrier indésirable examine les messages que vous envoyez et ajuste les filtres de courrier indésirable pour empêcher le courrier indésirable futur. Par conséquent, le courrier indésirable Office 365 filtre areconstantly affiné. Les éléments soumis sont évaluées à l’échelle du réseau. Envois positifs False sont passées en revue et évaluées d’ajustement de la règle possibles pour permettre aux futurs messages par les filtres de courrier indésirable. Par conséquent, il est utile pour vous et tous les clients qui utilisent le réseau global de notification au service de faux positifs et faux négatifs (non spam). L’équipe du courrier indésirable examine les indicateurs au sein de chaque message envoyé, telles que les suivantes :
  
- Adresse de l'expéditeur
    
- Adresse IP d'expédition
    
- Mots clés
    
- Expressions
    
- Fréquence de transmission
    
- Autres tendances et modèles
    
Une fois qu’ils passer en revue ces informations, l’équipe du courrier indésirable peut-être apporter des modifications à couches de filtrage du service. Pour plus d’informations sur l’équipe du courrier indésirable, vous pouvez regarder la vidéo suivante en anglais uniquement :
  
[Équipe Microsoft Exchange Spam vidéo](https://youtu.be/-TpX_-GMC7o?hd=1)
  
L’évaluation du courrier indésirable est un processus permanent qui s’applique quels que soient la langue et le jeu de caractères d’origine. Lorsque le message de courrier indésirable est vague ou manque de texte dans la ligne d’objet ou le corps, l’équipe de courrier indésirable s’appuie sur d’autres caractéristiques du message pour effectuer le filtrage. Cela signifie qu’après que l’équipe de courrier indésirable a balisé un message donné comme courrier indésirable et apporté les modifications nécessaires à sa base de règles, ce message reste bloqué jusqu’à ce que ses caractéristiques aient été suffisamment modifiées pour échapper à nos filtres. De nouvelles règles de courrier indésirable sont déployées en permanence. Les périodes pour les règles relatives aux soumissions individuelles dépendent de la quantité et de la qualité des soumissions. Comme les nouvelles règles de courrier indésirable sont définies globalement pour tous les utilisateurs, les diverses soumissions de courrier indésirable n’entraînent pas toutes l’introduction de nouvelles règles de courrier indésirable.
