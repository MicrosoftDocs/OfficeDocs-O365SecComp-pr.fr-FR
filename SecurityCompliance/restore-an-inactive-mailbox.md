## <a name="restore-an-inactive-mailbox"></a><span data-ttu-id="bb786-122">Restaurer une boîte aux lettres inactive</span><span class="sxs-lookup"><span data-stu-id="bb786-122">Restore an inactive mailbox</span></span>

<span data-ttu-id="bb786-p105">Utilisez la cmdlet **New-MailboxRestoreRequest** avec les paramètres  _SourceMailbox_ et  _TargetMailbox_ pour restaurer le contenu d'une boîte aux lettres inactive vers une boîte aux lettres existante. 1. <span data-ttu-id="bb786-125">Créez une variable contenant les propriétés de la boîte aux lettres inactive.</span><span class="sxs-lookup"><span data-stu-id="bb786-125">Create a variable that contains the properties of the inactive mailbox.</span></span> 
    
    ```
    $InactiveMailbox = Get-Mailbox -InactiveMailboxOnly -Identity <identity of inactive mailbox>
    ```

    > [!IMPORTANT]
    > <span data-ttu-id="bb786-p106">Dans la commande précédente, utilisez la valeur de la propriété **DistinguishedName** ou **ExchangeGUID** pour identifier la boîte aux lettres inactive. 2. <span data-ttu-id="bb786-p107">Restaurez le contenu de la boîte aux lettres inactive vers une boîte aux lettres existante.     ```
    New-MailboxRestoreRequest -SourceMailbox $InactiveMailbox.DistinguishedName -TargetMailbox newemployee@contoso.com -AllowLegacyDNMismatch
    ```
   
   <span data-ttu-id="bb786-p108">Vous pouvez également indiquer un dossier de premier niveau dans la boîte aux lettres cible, dans lequel restaurer le contenu de la boîte aux lettres inactive.     <span data-ttu-id="bb786-132">Cet exemple copie les éléments et sous-dossiers depuis une boîte aux lettres inactive vers un dossier intitulé « Boîte aux lettres inactive » dans la structure de dossiers de premier niveau de la boîte aux lettres cible.</span><span class="sxs-lookup"><span data-stu-id="bb786-132">This example copies mailbox items and subfolders from an inactive mailbox to a folder named "Inactive Mailbox" in the top-level folder structure of the target mailbox.</span></span>

   New-MailboxRestoreRequest -SourceMailbox $InactiveMailbox.DistinguishedName -TargetMailbox newemployee@contoso.com -TargetRootFolder "Inactive Mailbox" -AllowLegacyDNMismatch
   ```
