# <a name="detailed-properties-in-the-office-365-audit-log"></a>Propriétés détaillées dans le journal d’audit Office 365

Lorsque vous exportez les résultats d’une recherche de journal d’audit de la sécurité de 365 Office &amp; centre de conformité, vous avez la possibilité de télécharger tous les résultats qui correspondent aux critères de recherche. Pour ce faire, sélectionnez **Exporter les résultats** \> **Téléchargez tous les résultats** dans la page de **recherche des journaux d’Audit** de sécurité &amp; centre de conformité. Pour plus d’informations, voir [recherche dans le journal d’audit de sécurité Office 365 &amp; centre de conformité](search-the-audit-log-in-security-and-compliance.md).
  
 Lorsque votre exportation tous les résultats pour une recherche de journal d’audit, les données brutes à partir du journal d’audit unifiée est copié dans une virgule Office 365 séparés (fichier CSV) cela est téléchargé sur votre ordinateur local. Ce fichier contient des informations supplémentaires à partir de l’entrée du journal d’audit dans une colonne de **Détail**. Cette colonne contienne une propriété à valeurs multiples pour plusieurs propriétés de l’enregistrement du journal d’audit. Chacune des paires **: valeur de la propriété** dans cette propriété à valeurs multiples sont séparés par une virgule. 
  
Le tableau suivant décrit les propriétés qui sont incluses — selon Office 365 service dans laquelle un événement se produit, dans la colonne de **Détail** de propriétés multiples. La colonne **service Office 365 doté de cette propriété** indique le service et le type d’activité (utilisateur ou un administrateur) qui inclut la propriété. Pour plus d’informations sur ces propriétés ou sur les propriétés qui ne peuvent pas être répertoriées dans cette rubrique, voir [Office 365 gestion activité API Schema](https://go.microsoft.com/fwlink/p/?LinkId=717993).
  
> [!TIP]
> Vous pouvez utiliser la requête d’alimentation dans Excel pour fractionner cette colonne sur plusieurs colonnes afin que chaque propriété possède sa propre colonne. Cela vous permet de trier et filtrer sur un ou plusieurs de ces propriétés. Pour savoir comment procéder, voir la section « Fractionner une colonne par délimiteur » dans [une colonne de texte (requête Power) de fractionnement](https://support.office.com/article/5282d425-6dd0-46ca-95bf-8e0da9539662).
