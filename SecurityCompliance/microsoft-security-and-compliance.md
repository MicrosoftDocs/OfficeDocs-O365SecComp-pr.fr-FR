# <a name="get-ready-for-the-new-microsoft-365-security-center-and-microsoft-365-compliance-center"></a>Préparez-vous pour le nouveau centre de sécurité Microsoft 365 et le centre de conformité 365 Microsoft

**Nous avons le plaisir de vous annoncer le [Centre de conformité Microsoft 365](#microsoft-365-compliance-center), présentant commençant à la fin de janvier 2019 et par le biais de mars 2019 et tout nouveau [Centre de sécurité Microsoft 365](#microsoft-365-security-center) **. Lisez cet article pour obtenir une vue d’ensemble des nouveautés, [à quoi s’attendre](#what-to-expect)et [licences et autorisations nécessaires](#required-licenses-and-permissions).

## <a name="microsoft-365-security-center"></a>Centre de sécurité Microsoft 365

Votre nouveau centre de sécurité Microsoft 365 comprend un tableau de bord pour vous aider à gérer et surveiller la sécurité entre les identités, périphériques, applications, infrastructure et les données. Vous serez également accéder facilement à votre Score de sécuriser Microsoft toute nouvelle, des rapports de menace de périphérique, des rapports de menace d’identité et vos rapports de sécurité des applications dans le nuage. 

![Nouveau centre de sécurité Microsoft 365](media/m365-security-center.png)

Lorsque vous utilisez le centre de sécurité Microsoft 365 pour la première fois, les informations s’affichent dans la partie supérieure de l’écran pour vous aider à démarrer. Vous verrez également comment accéder facilement aux fonctionnalités de sécurité qui vous intéressent exploration.

Une fois que le centre de sécurité Microsoft 365 est activé pour votre client, vous serez en mesure d’accéder à l’adresse [https://security.microsoft.com](https://security.microsoft.com). 

> [!NOTE]
> Vous devez être affecté à un rôle d’Azure Active Directory valide pour accéder au centre de sécurité Microsoft 365. Pour plus d’informations, consultez la section [licences et autorisations requis](#required-licenses-and-permissions) (dans cet article).

## <a name="microsoft-365-compliance-center"></a>Centre de conformité 365 Microsoft

Votre nouveau centre de conformité 365 Microsoft vous propose de visibilité en conformité Gestionnaire Microsoft, qui reflète votre état de conformité globale et vous donne les mesures recommandées pour vous aider à configurer les paramètres pour répondre aux obligations de conformité complexes. Vous devez accéder facilement aux étiquettes de sensibilité et de rétention et stratégies, prévention des pertes de données (DLP), la gouvernance des données, eDiscovery, demandes d’objet de données (de manière anonyme), gestion et la sécurité d’application dans le nuage. En outre, vous pouvez obtenir des informations pratiques et tirer parti d’automatisation intelligente pour réduire les risques de conformité et de protéger votre immobiliers numériques. 

![Centre de conformité 365 Microsoft](media/m365-compliance-center.png)

Lorsque vous utilisez le centre de conformité 365 Microsoft pour la première fois, les informations s’affichent dans la partie supérieure de l’écran pour vous aider à démarrer. Vous allez découvrir comment accéder facilement aux fonctionnalités de conformité qui vous intéressent exploration.

Une fois le centre de conformité 365 Microsoft est activé pour votre client, vous serez pour accéder à l’adresse [https://compliance.microsoft.com](https://compliance.microsoft.com).  

> [!NOTE]
> Vous devez être affecté à un rôle d’Azure Active Directory valide pour accéder au centre de conformité Microsoft 365. Pour plus d’informations, consultez la section [licences et autorisations requis](#required-licenses-and-permissions) (dans cet article).

## <a name="what-to-expect"></a>À quoi s’attendre

### <a name="coming-soon"></a>Bientôt disponible

Le nouveau centre de sécurité Microsoft 365 et le nouveau centre de conformité Microsoft 365 sont propagées commençant fin janvier et par le biais de 2019 mars. Vous devriez avoir accès au centre de conformité Microsoft 365 pendant cette période et nouveau centre de sécurité Microsoft 365.

### <a name="easy-access"></a>Accès facile !

Avec l’amélioration de la navigation, des solutions intégrées et une expérience simplifiée, vous serez en mesure de voir et d’accéder aux informations vous intéressent et tirez parti de la sécurité puissante et solutions de conformité dans Microsoft 365.

### <a name="smooth-transition"></a>Transition en douceur !

Vous pouvez vous attendre une transition en douceur vers les centres de nouveau. Une fois que cette modification est entièrement déployée, nous envisagez de retirer l’ancien Microsoft 365 Security & centre de conformité ([https://protection.microsoft.com](https://protection.microsoft.com)). Expérience de l’administrateur va changer, mais cela n’affecte pas vos configurations de conformité et de sécurité.

Une fois cette mise à jour est déployée, si votre organisation dispose de Microsoft 365 entreprise E3 ou E5, les administrateurs de sécurité et de conformité peuvent :

- Accédez directement à [https://security.microsoft.com](https://security.microsoft.com) et [https://compliance.microsoft.com](https://compliance.microsoft.com); <br>ou  
- Accédez au centre d’administration Microsoft 365, puis accédez au nouveau centre de sécurité Microsoft 365 et centre de conformité Microsoft 365 (liens sont sous centres d’administration dans le volet de navigation de gauche).

> [!TIP]
> Si vous utilisez le centre de conformité de & Office 365 Security ([https://protection.office.com](http://protection.office.com)), vous pourrez toujours configurer et gérer vos paramètres Office 365 au sein de votre & de sécurité pour Microsoft Office 365 existant centre de conformité. Configurations seront conservées dans l’existant Office 365 sécurité & centre de conformité ainsi que dans le nouveau centre de sécurité Microsoft 365 et le centre de conformité Microsoft 365.  

## <a name="required-licenses-and-permissions"></a>Autorisations et les licences requises

Pour obtenir le nouveau centre de sécurité Microsoft 365 et le centre de conformité Microsoft 365, votre organisation doit disposer d’un abonnement à Microsoft 365 E3 ou E5 ou une licence en Volume équivalent (qui se compose d’Office 365 entreprise E3 ou E5, mobilité d’entreprise + sécurité E3 ou E5 et Windows 10 entreprise E3/E5).

Le rôle Administrateur général, administrateur de conformité, administrateur de sécurité ou lecteur de sécurité dans Azure Active Directory pour accéder au nouveau centre de sécurité Microsoft 365 ou un centre de conformité Microsoft 365 doit être assignée aux utilisateurs.

- Les administrateurs globaux peuvent accéder à la fois le centre de sécurité et le centre de conformité

- Les administrateurs de conformité peuvent accéder au centre de conformité

- Les administrateurs de sécurité ou de sécurité lecteurs peuvent accéder au centre de sécurité

> [!NOTE]
> Rôles supplémentaires, notamment l’opérateur de sécurité et administrateur de données de conformité, sont seront prochainement disponibles.
