## <a name="how-it-works"></a><span data-ttu-id="20f8b-111">Fonctionnement</span><span class="sxs-lookup"><span data-stu-id="20f8b-111">How it works</span></span>

Although Chris can see the file is there, Chris cannot open or share it, which prevents Chris's computer and others from the malicious file.</span></span>
