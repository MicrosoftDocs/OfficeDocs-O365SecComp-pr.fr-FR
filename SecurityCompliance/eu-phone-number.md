# <a name="eu-phone-number"></a>Numéro de téléphone de l’Union européenne

Prévention des pertes de données (DLP) de sécurité Office 365 &amp; centre de conformité inclut de nombreux types d’informations sensibles qui sont prêts à utiliser dans vos stratégies DLP. Cette rubrique décrit le type d’informations sensibles de numéro de téléphone de l’UE. 
  
## <a name="austria"></a>Autriche

### <a name="format"></a>Format

Aucune longueur standard
  
### <a name="pattern"></a>Modèle

- Chiffres 
    
- Uniquement les numéros de téléphone mobile commencent par le chiffre « 6 » 
    
### <a name="checksum"></a>Somme de contrôle

Non applicable
  
### <a name="definition"></a>Définition

Le pourcentage de confiance d’une stratégie DLP ayant détecté ce type d’informations sensibles est de 75 % si, dans une proximité de 300 caractères :
  
- L’expression régulière `Regex_austria_eu_telephone_number_1` recherche de contenu qui correspond au modèle. 
    
- Un mot clé à partir de `Keywords_austria_eu_telephone_number` est trouvée. 
    
Le pourcentage de confiance d’une stratégie DLP ayant détecté ce type d’informations sensibles est de 75 % si, dans une proximité de 300 caractères :
  
- L’expression régulière `Regex_austria_eu_telephone_number_2` recherche de contenu qui correspond au modèle. 
    
- Un mot clé à partir de `Keywords_austria_eu_telephone_number` est trouvée. 
    
